# NL_Visuals

NL Visual photography — a Next.js photography portfolio, deployable to Vercel.

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deploying to Vercel

This project is configured for zero-config deployment on [Vercel](https://vercel.com/).

1. Push this repository to GitHub.
2. Import the project in the [Vercel dashboard](https://vercel.com/new).
3. Vercel will auto-detect the Next.js framework and deploy.

Or deploy instantly with the button below:

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https%3A%2F%2Fgithub.com%2FJonSvitna%2FNL_Visuals)
